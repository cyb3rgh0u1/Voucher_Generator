# Voucher Generator Script

## Author
CyberGhoul (cyb3rgh0u1)

## Description
This script generates unique vouchers with a specified limit on the number of vouchers that can be generated. It was initially a test project, and it is free to use and modify. If you use this code, please give credit to the author, cyb3rgh0u1.

## Features
- Generates unique vouchers consisting of uppercase, lowercase letters, and digits.
- Allows the user to specify the voucher length.
- Limits the number of vouchers generated based on user input.

## Usage
1. **Download the Script**: Save the script to your local machine.
2. **Run the Script**: Execute the script using Python.
3. **Input Parameters**:
    - `VOUCHER_LIMIT`: The number of vouchers to generate.
    - `VOUCHER_LENGTH`: The length of each voucher.

### Example
```bash
python3 voucher_generator.py
```

When prompted, enter the desired voucher limit and length:
```
VOUCHER_LIMIT = 10
VOUCHER_LENGTH = 8
```
The script will generate and print the specified number of vouchers, each of the specified length.

Sample Output
```
f5G7h8K3
d2L5k0N8
j9P4m7Q1
........
```
